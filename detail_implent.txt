Detalhes da Implementação
API com Flask: Utilizamos Flask para criar endpoints que processam os dados e retornam os resultados das previsões feitas pelos modelos de machine learning.
Modelos de Machine Learning: Simulamos um modelo de regressão linear simples usando scikit-learn para prever dados futuros com base em um conjunto de dados fictício.
Visualizações Interativas: Usamos a biblioteca Plotly para gerar gráficos interativos que exibem as previsões.
Integração Front-end e Back-end: O front-end envia as entradas do usuário para o back-end usando AJAX, e as previsões são renderizadas dinamicamente no navegador.
Como Executar o Projeto
Configure o Ambiente: Certifique-se de ter Python e os pacotes necessários instalados.

Execute o Servidor Flask:

bash
Copy code
python app.py
Acesse o Site: Abra um navegador e vá para http://localhost:5000 para ver a página de exemplos de machine learning interativos.
